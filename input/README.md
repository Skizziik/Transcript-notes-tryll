# input/

Кидай сюда **ровно один** аудиофайл (m4a, mp3, wav, mp4, mov, opus, aac, flac, aiff, ogg, webm).

Запусти `/notes` в Claude Code — он подхватит файл, транскрибирует и сложит заметки в `../output/<имя>_<дата>/<имя>.notes.docx`.

Если файлов больше одного — команда остановится и попросит убрать лишние. Старые аудио удаляй сам.
